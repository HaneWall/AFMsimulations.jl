AFMsimulations.Φ_poincare_fixpoint — Method

Poincare map Φ: x(t) –> x(t + T), with T being the period of the forcing term. For us T = 2π * Ω. Input is a ODE-Problem, initial Vector x, problem specific parameters and Δt. We found a periodic orbit iff |Φ(x) - x|_2 <= ϵ with ϵ being the absolute tolerance. This function can tzhan be used to determine a callback function in an ODEProblem. This way we Should be able to get way more efficient algorithms.
